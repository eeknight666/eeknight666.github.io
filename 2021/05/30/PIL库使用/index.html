PIL库使用

PIL库使用

函数说明

Image.open()

读取文件

Image.save()

保存文件，也可以用save方法来进行图片的格式转换。

new

方法使用给定的模式和大小创建一个新图像

这里通常和paste组合使用

format

format是格式，

size

size 是一个元组,表示(宽，高)

mode

mode则指的图片的模式

show()

这个类似于直接打开图片

os.path.splitext()

这个是os模块中的方法，可以将文件名和扩展名分离开来

crop()方法。

从一张图片中剪切出一块区域，比如从图片提取矩形。

transpose()

Transpose 方向变换

有一下几种模式FLIP_LEFT_RIGHT ，FLIP_TOP_BOTTOM ，ROTATE_90 ，ROTATE_180 ，ROTATE_270，TRANSPOSE ，TRANSVERSE。

rotate()

旋转图片

paste()

粘贴，也就是将图片部分地方修改

格式为：paste(要贴的图片，要贴的图片的4元坐标组成的区域)。

如下面，我们把xiao.jpg这张图片，取区域(30,30,80,100)，将该区域旋转180度后贴到原来的位置。

getpixel

函数是用来获取图像中某一点的像素的RGB颜色值，getpixel的参数是一个坐标点。对于图象的不同的模式，getpixel函数返回的值有所不同。

返回的是坐标点(x,y)处的red，green，blue的数值。

putpixel

putpixel()修改x，y处的像素。对于单波段图像，颜色作为单个数值给出；对于multi-band图像，颜色作为元组给出

上面是修改坐标的red和green值，blue按原来的值不改变

gif图片排序

虽然可以用工具提取，但是还是要学一下PIL的方法

当处理GIF这种包含多个帧的图片，称之为序列文件，PIL会自动打开序列文件的第一帧。而使用seek和tell方法可以在不同帧移动。tell是帧数，而seek是取当前帧数的图片。

while

for

如果要使用for循环，可以使用ImageSequence模块的Iterator方法。

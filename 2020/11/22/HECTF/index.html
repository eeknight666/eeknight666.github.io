前言

这个比赛难度还可以，虽然我是菜鸡，第一天还是解了几题，然后第二天啥都不会，然后web还被砍分了，导致排名跌了很多，心累。

web

签到

抓包，然后看到右下串数字，猜测是手机号

然后到忘记密码这里，说是4位数字，对他进行爆破

得到验证码0233（这个验证码是固定的）

修改密码后再登录就得到flag

ezphp

源码

解题思路：parm1！==parm2 md5要相等，就数组绕过

第二个是一个替换，所以要构造一个一cxhp和数字组成的md5，还要以ce开头

用脚本就可以找到

payload：?str1=240610708&str2=9427417&param1[]=1&param2[]=2

ssrf

解题思路：传入的不能是127.0.0.1/10.0.0.0/172.16.0.0/192.168.0.0这几个

所以要绕过，也就是换种方式让他指向上面几个地址中的一个

这里用127.0.0.1==http://[0:0:0:0:0:ffff:127.0.0.1]/

payload：http://114.55.165.246:12003/?url=http://[0:0:0:0:0:ffff:127.0.0.1]/flag.php
